# **App Name**: NovaRoof Solutions Landing

## Core Features:

- Informative Home Page: A comprehensive landing page detailing NovaRoof Solutions LLC's services, expertise, and contact information.
- About Us Page: A dedicated 'About Us' page with well-structured information about the company's history, mission, and values.
- Services Section: A detailed section outlining all roofing-related services offered by NovaRoof Solutions LLC.
- Contact Form: Allow users to send emails directly from the landing page.
- WhatsApp Integration: A floating WhatsApp button that links to WhatsApp Web or the WhatsApp app, facilitating direct communication.
- Footer with Legal Information: A well-structured footer containing links to the company's privacy policy, cookie policy, and terms and conditions.

## Style Guidelines:

- Primary color: Dark red (#8B0000), reflecting strength and reliability.
- Background color: Light gray (#D3D3D3), offering a clean and modern backdrop.
- Accent color: White (#FFFFFF) to highlight key information and calls to action.
- Body text: 'Inter', a sans-serif font that looks modern and machined, and works well for longer paragraphs
- Headlines: 'Space Grotesk', a sans-serif font suitable for headlines.
- Employ a grid-based layout for well-organized information and clear visual hierarchy.
- Use simple, solid icons related to roofing services for visual clarity.
